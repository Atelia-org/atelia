## Action

**Virtual-Sense**：虚拟身体的第一系统，疼痛、饥饿、疲劳、快感、紧张等。`伤口钻心地疼。`
**Thinking**：一般思维链。`我应该振作起来。`
**Feeling**：心理感觉，是一种认知。
**Monologue**：自己对自己说的话。`“Galatea 你要振作起来！”`。
**Decision**：决策。
**Intent**：意图。
**Action**：立即执行的具体动作。
**String-Out**：说、写、发信息。配合Target修饰处理复杂情况。
