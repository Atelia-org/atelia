# term `Document-Root` 文档根节点

This term defines the root concept.

## decision [ROOT-REQUIREMENT] 根级决策

All documents must have a root.

### spec [ROOT-FORMAT] 根格式规范

The root format follows markdown convention.

#### derived [ROOT-EXAMPLE] 根节点示例

This is an example of a root node.

## term `Child-Concept` 子概念

A child concept nested under the root.

### spec [CHILD-RULE] 子节点规则

Children must follow parent constraints.

## decision [GLOBAL-CONSTRAINT] 全局约束

This is a global constraint at level 2.
